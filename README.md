# Chitralekha Boutique Resort — Static Website

A premium static marketing site for **Chitralekha Boutique Resort**, Dehradun.

## Hero video background

The hero uses your Google Drive resort video. For **best autoplay**, download it from
[Google Drive](https://drive.google.com/file/d/11qABCDoANR-fGDmuU_h49mDoW3Vs2VGY/view) and save as `videos/resort-hero.mp4`.

## Quick start

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Customize for your client

1. **Phone & email** — Update links in the Contact section (`index.html`) and the form recipient in `main.js` (`mailto:` address).
2. **Photos** — Resort images live in `images/` (10 photos with captions in HTML).
3. **Google Maps** — The embed uses the resort’s place ID; verify it loads correctly or paste a fresh embed from [Google Maps](https://www.google.com/maps/place/Chitralekha+Boutique+Resort+Dehradun).

## Deploy

Upload all files to any static host: Netlify, Vercel, GitHub Pages, or your client’s hosting.
